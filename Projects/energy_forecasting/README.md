# Energy Forecasting using Machine Learning

## Objective

Build a machine learning system to forecast energy demand using historical time-series data and engineered features.

## Methods

* Feature engineering on time-series data (lags, rolling statistics)
* Machine learning models: XGBoost, Random Forest
* Baseline comparison with statistical approaches
* Evaluation using MAE and RMSE

## Workflow

1. Data preprocessing
2. Feature engineering
3. Train/test split (time-based)
4. Model training
5. Evaluation and error analysis

## Planned Extensions

* GRU/LSTM deep learning models
* External features (weather, calendar effects)
* Probabilistic forecasting

## Tools

Python, Pandas, Scikit-Learn, XGBoost
