# Machine Learning Forecasting Portfolio

This repository contains applied machine learning projects focused on forecasting, energy systems, and geospatial analytics.

## Focus Areas

* Time-series forecasting for energy and infrastructure systems
* Spatiotemporal modeling using geospatial and mobility data
* Machine learning models for demand prediction and system optimization

## Projects

### 1. Energy Forecasting

Machine learning models for predicting energy demand using time-series and external features.

### 2. EV Charging Forecasting

Forecasting EV charging station utilization using spatiotemporal and behavioral data.

### 3. Geospatial ML

Spatial data analysis and geospatial feature engineering for urban infrastructure systems.

## Tech Stack

Python, Pandas, NumPy, Scikit-Learn, XGBoost, PyTorch

## Goal

To build production-oriented machine learning systems for real-world forecasting problems in energy and infrastructure domains.
